# miniEcommerce
* mini-ecommerce com carrinho de compras e formulário de dados com validação

## dependências
* react
* bootstrap
* react-datepicker: para edição da data do formulario

* formik e yup: para formulario
* axios: requisiçoes http
* servidor-api no backend
